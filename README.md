# 또다른 단젤
node.js를 이용한 개인용 디스코드 봇.  

## 설치 & 실행
```javascript
npm i //패키지 다운로드
config.json //파일 생성  
node app.js //봇 실행
```
**config.json**

```json
{
    "token": "디스코드 봇 토큰",
}
```

## 사용법
*prefix 변수 변경을 통해 명령어 접두사 변경 가능

#### 단젤아 <노래 제목>
유튜브에서 노래 제목 검색 후 재생

#### 단젤아 그만
재생중인 노래 중단

#### \` <메세지>
TTS로 메세지 출력
